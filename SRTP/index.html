Welcome to the StepTrack

Here's a quick guide to get you started:
  1. Open the excel regression document you're using.
  2. Go to whichever tab you'd like to test.
  3. Copy the entirety of the testing steps. You can include the number to the left of each step if you'd like. Helps with organization.
  4. Paste them into a blank text file. Ensure that each step is on its own line (your computer should do this automatically).
  5. Save the text file (can be named anything - just remember where you saved it).
  6. Click the "Choose File" button above and select the file.
  7. Enter the name of the client you're testing for.
  8. Enter the name of the tab you'll be testing.
  9. Click "Begin" to start the program.
