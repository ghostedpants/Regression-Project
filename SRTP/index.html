Welcome to the Super Regression Testing Project. Here's a quick guide to get you started.

  1. Open the excel regression document you're using.
  2. Go to whichever tab you'd like to test.
  3. Copy ONLY the testing steps from the tab (not the column names at the top, or the numbers behind the steps).
  4. Paste them into a blank text file.
  5. Click the "Choose File" button above and select the file.
  6. Click "Begin" to start the program.
